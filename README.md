# 🐦 Flappy Bird (C++ & Raylib)

A solid, fully playable clone of the classic Flappy Bird game, built using C++ and the Raylib graphics library. This project demonstrates Object-Oriented Programming (OOP) principles, real-time game loops, physics simulation, and 2D rendering.

## ✨ Key Features

* **Object-Oriented Design:** Built using custom C++ classes (e.g., Bird, Pipes) to ensure clean, modular, and maintainable code.
* **Physics & Gravity Simulation:** Smoothly simulates the bird's falling momentum and jump arcs using real-time frame updates.
* **Dynamic Obstacle Generation:** Pipes are generated at random heights, requiring quick reflexes and creating an endless gameplay loop.
* **Real-time Collision Detection:** Accurately detects when the player hits the ground, the ceiling, or the incoming pipes using Raylib's 2D bounding boxes.
* **Score Tracking:** Keeps track of the player's current score as they successfully navigate through the gaps.

## 🎮 How to Play

1. Run the compiled executable or build the Visual Studio Solution (`.sln`).
2. Press **[Spacebar]** (or the Left Mouse Button) to make the bird flap and jump.
3. Avoid hitting the pipes, the ground, or the screen borders.
4. Survive as long as possible to get the highest score!

## 🛠️ Technologies & Tools Used

* **Language:** C++ (OOP Paradigm)
* **Graphics Library:** [Raylib](https://www.raylib.com/) (For 2D rendering, window management, and input handling)
* **Environment:** Microsoft Visual Studio
